# 📋 QR Menu Web Application

This project is a simple frontend application created to present a restaurant menu on the web. The menu data is fetched from a `db.json` file and users are redirected to a detail page. The application is developed using Vanilla JavaScript, HTML, CSS, and Bootstrap.

---

# 🔧 Features

🔘 Filtering menu items by category (Breakfast, Lunch, Beverages)

🔍 Redirecting to detail page (using query parameter: detail.html?id=1)

📱 Responsive design (using Flexbox and CSS Grid for layout)

🔗 Data management with json-server

---

# 🧰 Technologies Used

- HTML5

- CSS3

- Bootstrap 

- JavaScript (ES6 Modules)
